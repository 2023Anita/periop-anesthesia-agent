# Code of Conduct

This project is a small clinical AI developer template. We want discussions and contributions to be respectful, specific, and safety-aware.

## Expected Behavior

- Be respectful and constructive.
- Keep medical claims conservative.
- Discuss clinical safety boundaries clearly.
- Use synthetic data only in examples, tests, screenshots, and demos.
- Prefer small, reviewable contributions.

## Unacceptable Behavior

- Harassment, discrimination, or personal attacks.
- Sharing real patient data.
- Encouraging unsafe medical use of this prototype.
- Presenting the project as a production medical device.
- Adding autonomous diagnosis, dosing, emergency instructions, surgery clearance, or patient-specific advice.

## Reporting

Open a GitHub issue if a project interaction or contribution creates a safety concern. For private concerns, contact the maintainer through the public GitHub profile.

Maintainers may edit, close, or reject contributions that violate these boundaries.
